# Wi-Fi Security Analyzer 🔐

# Overview
A Python-based tool that passively analyzes Wi-Fi networks to detect vulnerabilities such as weak encryption, unauthorized devices, and potential rogue access points.

# Features
- Network scanning (SSID, signal, encryption)
- Packet sniffing using Scapy
- Security risk analysis
- Evil twin detection
- Unauthorized device detection

# Tech Stack
- Python
- Scapy
- PyWiFi

# Disclaimer
This tool is for educational and ethical use only.
